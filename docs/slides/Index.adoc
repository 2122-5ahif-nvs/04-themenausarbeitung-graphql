=== Underfetching

Ein Request bringt nicht alle nötigen Daten zurück +
-> mehrere Requests notwendig

[background-color="#2c2d89"]

=== Overfetching

Ein Request bringt zu viele Daten zurück +
-> Daten werden umsonst verschickt

[background-color="#2c2d89"]
